# Input Mapping - Controller Visualization

A modern web application for visualizing controller input mappings.

## Features
- **Visual Feedback**: Real-time visualization of controller buttons.
- **Support for multiple controllers**: Xbox, PlayStation, and Switch layouts.
- **Dynamic Lines**: Smooth SVG curves connecting input fields to controller buttons.
- **Persistent Storage**: Save your mappings locally in the browser.

## Technologies
- HTML5 / CSS3 (Vanilla)
- Vanilla JavaScript
- SVG for high-quality controller icons
- Vercel for deployment

## Deployment
This project is configured for one-click deployment to Vercel.

1. Push this project to GitHub.
2. Link your GitHub repository to [Vercel](https://vercel.com).
3. Vercel will automatically detect the configuration and deploy your app.
